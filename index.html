平 台 說 明

經書解鎖人生是一個結合千年佛典智慧與現代 AI 的心靈引導平台。

使 用 方 式

  1. 從首頁選擇一部經書（金剛經或心經）
  2. 在文字框描述您正面對的困境或煩惱
  3. 點按「求解」，系統將從經典中找出最相關的經文與解析
  4. 也可點選下方的「常見困境」快速分類，或按「隨機困境」體驗
  5. 設定 OpenAI API Key 後，可啟用 ChatGPT 個人化深度解惑
  6. 點按「朗誦」按鈕可聆聽完整經文的語音朗讀

金剛般若波羅蜜經 · 姚秦三藏法師鳩摩羅什譯

The Diamond Sutra · Translated by Kumārajīva, 402 CE

平 台 說 明

經典解鎖人生是一個結合千年佛典智慧與現代 AI 的心靈引導平台。

使 用 方 式

  1. 從首頁選擇一部經書（金剛經、心經、阿彌陀經或妙法蓮華經）
  2. 在文字框描述您正面對的困境或煩惱
  3. 點按「求解」，系統將從經典中找出最相關的經文與解析
  4. 也可點選下方的「常見困境」快速分類，或按「隨機困境」體驗
  5. 設定 Gemini 或 OpenAI API Key 後，可啟用 AI 個人化深度解惑
  6. 點按「朗誦」按鈕可聆聽完整經文的語音朗讀

AI 功 能

支援 Google Gemini（免費）與 OpenAI ChatGPT 雙引擎。
兩者均可同時設定，點按「⚙ AI 設定」選擇使用哪個 AI。
API Key 僅儲存於您的瀏覽器本地，不會上傳至任何伺服器。
